SendGrid Template Translation

This is a SendGrid to SparkPost template translation service. From here you can translate a template from file or migrate one directly from your SendGrid account into SparkPost.

MIGRATE A TEMPLATE » TRANSLATE TEMPLATE TEXT »

Supported Features

  • Sections
    • <%subject%>, <%body%>
    • [Sender_Name], [Sender_Name], [Sender_Address], [Sender_City], [Sender_State], [Sender_Zip]
    • [Weblink]
    • [%email%]
    • [%custom_field%], [%custom_field | default value%]
  • Needs extra info (Links after translation is done)
    • <%asm_global_unsubscribe_url%>
    • [Unsubscribe]
  • User delimited variables
    You need to specify the custom delimiters that you've used in your template.

Unsupported Features

  • <%asm_group_unsubscribe_url%>
  • <%asm_preferences_url%>
  • [Unsubscribe_Preferences]
