SendGrid Template Translation

This is a SendGrid to SparkPost template translation service. From here you can translate a template from file or migrate one directly from your SendGrid account into SparkPost.

MIGRATE A TEMPLATE » TRANSLATE TEMPLATE TEXT »

Transactional Templates

Supported features Unsupported features
  • <%subject%>
  • <%body%>
  • <%asm_global_unsubscribe_url%>
    Note: You'll need to replace ? in <a href="?" data-msys-unsubscribe="1">unsubs</a> after translation.
  • Custom delimited variables
    You need to provide the custom delimiters during translation.
    Only one type of delimiter is supported.
  • <%asm_group_unsubscribe_url%>
  • <%asm_preferences_url%>
